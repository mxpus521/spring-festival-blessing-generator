# 【新春创造营】春节祝福语生成器

## 🎉 项目介绍

大家好！在2026年春节来临之际，我使用 iFlow CLI 创建了一个实用的春节祝福语生成器！

这个项目解决了很多人"过年不知道说什么"的痛点，提供5大主题、165条精选祝福语，帮助用户快速生成个性化祝福语，一键复制，方便分享给亲朋好友。

## ✨ 使用功能

### 🎯 5大主题可选
- **拜年祝福**：传统春节祝福，适合长辈、同事
- **职场祝福**：事业升职、工作顺利，适合职场人士
- **亲情祝福**：阖家欢乐、健康平安，适合家人
- **爱情祝福**：甜蜜浪漫、天长地久，适合情侣
- **趣味祝福**：幽默风趣、轻松愉快，适合朋友

### 🎨 核心功能
1. **一键生成**：选择主题后点击生成，智能随机祝福语
2. **打字机效果**：祝福语逐字显示，增加趣味性
3. **一键复制**：复制按钮快速复制到剪贴板
4. **标签分类**：每条祝福语带标签，方便识别
5. **防重复机制**：智能避免短时间内重复生成

### 🎬 视觉效果
- 中国红 + 金色主题配色，节日氛围浓厚
- CSS绘制灯笼装饰，带摆动动画
- 按钮发光效果，增加交互感
- 流畅的动画效果，提升用户体验

### 📱 响应式设计
完美适配桌面端、平板端、手机端，随时随地使用！
